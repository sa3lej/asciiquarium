# Changelog

## [0.1.0] - 2026-04-20

### Added
- Complete rewrite in Rust (previously Perl)
- Crossterm-based terminal rendering
- Animated sea creatures: fish, sharks, dolphins, whales, ducks
- Castle, seaweed, and bubble animations
- Image-to-ASCII art rendering (--image flag)
- Keyboard controls (arrow keys, +/- speed, p pause, q quit)
- Classic mode (--classic) for original color palette
- Full color support with 256-color terminal
- Command-line options via clap

### Removed
- Original Perl source code
- Legacy MANIFEST and CHANGES files

### Changed
- Complete architecture rewrite for performance and cross-platform support
